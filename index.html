My Github repositories

https://github.com/srishina/delayq.git : Delayed queue in go.

DelayQueue is a synchronized delay queue which is similar or a specialized form of a priority queue. The queue orders the elements based on their delay time. Only the expired elements can be retrieved from the queue.

https://github.com/srishina/mqtt.ts.git : Typescript library for the MQTTv5 protocol.

mqtt-ts is a client library for the MQTTv5 protocol, written in typescript for node and browser. The library uses promises and async/wait patterns.

https://github.com/srishina/mqtt.go.git : Go implementation of MQTTv5 protocol.

MQTTv5 client and server library that implements MQTTv5 protocol.
